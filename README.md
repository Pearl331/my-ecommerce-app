A full-stack e-commerce web application built using the MERN (MongoDB, Express.js, React, Node.js) stack. This project allows users to browse products, filter by category, add items to a cart, and complete a selective checkout process.

🚀 Key Features
Product Catalog: Fetches 100+ products using a public API.

Smart Filtering: Filter products by categories and search by name.

Shopping Cart: Add/remove items with state management.

Checkout System: Selective checkout where users provide shipping details.

Data Persistence: Orders are saved permanently to a MongoDB database.

🛠 Tech Stack
Frontend: React, React Router, Axios, CSS.

Backend: Node.js, Express.js.

Database: MongoDB, Mongoose.

⚙️ How to Run Locally
Prerequisites
Node.js installed on your computer.

MongoDB Compass installed.
